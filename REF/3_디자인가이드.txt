### 디자인 컨셉트
**"Warm Editorial SaaS"** — 따뜻하고 친근하면서도 세련된 교육 플랫폼. Framer, Lemon Squeezy처럼 사람 냄새가 나는 모던 SaaS 미학을 지향합니다. 유치하거나 과하지 않고, 신뢰감과 접근성을 동시에 전달합니다. 잡지식 에디토리얼 레이아웃으로 스크롤이 자연스럽게 이어지며, AI 교육 플랫폼임을 직관적으로 느낄 수 있도록 설계했습니다.

### 그레이스케일
순수 흑백 그레이스케일을 사용하지 않고, **모든 중립 톤에 따뜻한 브라운/베이지 계열을 혼합**합니다. 차가운 회색(`#888888`) 대신 `#8C7B6E`처럼 황색 기운이 도는 웜 뉴트럴을 사용해 전체 페이지가 일관되게 따뜻한 온도감을 유지합니다. 경계선(border)도 `rgba(30,22,18,0.10)` — 투명도 있는 딥 브라운으로 처리해 딱딱한 느낌 없이 자연스럽게 구분합니다.

### 시각요소
- **SVG 일러스트 3종 직접 제작** — 히어로 코드 에디터 창, 소개 섹션 인물 일러스트 (사람이 컴퓨터 앞에 앉아있는 장면), 기대효과 수료 전/후 비교 바 차트
- **플로팅 배지** — "즉시 생성 ⚡", "바로 배포 🚀", "AI 디버깅 🤖" 등 SVG 위에 떠 있는 작은 카드 형태의 정보 배지
- **기술스택 플로우 다이어그램** — 아이디어 → AI 코딩 → 코드 완성 → 테스트 → 배포 완료를 점선 화살표로 연결한 SVG 플로우
- **마키 스트립** — 기술 키워드가 좌우로 흐르는 무한 스크롤 텍스트 배너
- **배경 블롭** — 히어로와 CTA 섹션에 흐릿한 방사형 그라디언트 오브를 배치해 깊이감을 부여. 직접적인 이미지 없이 분위기만 전달
- **아이콘** — 이모지 기반 아이콘 사용으로 친근한 톤 유지. 외부 아이콘 라이브러리 불필요

---

### 레이아웃
- **컨테이너**: `max-width: 1600px`, `margin: 0 auto`, 좌우 패딩 `60px`
- **헤더**: 브라우저 100% 너비 고정(`position: fixed`), 내부 컨텐츠만 1600px 정렬
- **섹션 패딩**: 상하 `120px` 일관 적용으로 충분한 여백 확보
- **그리드 시스템**
  - 히어로 / 소개 / 기대효과: `1fr 1fr` 2컬럼 좌우 분할
  - 커리큘럼: `repeat(4, 1fr)` 4컬럼 카드
  - 커뮤니티: `repeat(3, 1fr)` 3컬럼 카드
  - 기술스택: 카테고리별 `flex-wrap` 태그 나열
- **반응형**: 960px 이하 1컬럼 전환, 640px 이하 모바일 최적화
- **글씨 절대 안 잘림**: 고정 height 사용 금지, 모든 텍스트 컨테이너 `min-height` 또는 자동 높이 적용

---

### 사용자 경험(UX) 요소
- **헤더 스크롤 감지** — 스크롤 시 반투명 배경 + `backdrop-filter: blur` 적용으로 콘텐츠와 자연스럽게 분리
- **스크롤 리빌 애니메이션** — `IntersectionObserver`로 뷰포트 진입 시 `fadeUp` (opacity + translateY) 순차 등장. 섹션 내 카드마다 `d1`~`d5` delay 클래스로 스태거 효과
- **카드 호버** — 모든 카드에 `translateY(-4px to -6px)` + `box-shadow` 강화로 들어올리는 느낌
- **커리큘럼 카드 호버** — 하단 코랄 바(`curr-bar`)가 `scaleX(0 → 1)`로 드로우-인
- **기대효과 리스트 호버** — `padding-left` 증가로 미끄러지듯 들여쓰기
- **기술스택 태그 호버** — 코랄 틴트 배경 + 보더 컬러 변경 + `translateY(-2px)`
- **버튼 shimmer** — CTA 버튼에 빛이 가로질러 가는 글로우 애니메이션
- **히어로 텍스트** — 순차 `animation-delay`로 배지 → 타이틀 → 설명 → 버튼 → 통계 순서로 등장
- **GNB** — 수강신청 버튼을 코랄 색상으로 강조해 전환 유도. 나머지 링크는 뮤트 컬러로 시각 위계 설정

### 추가 요청사항
- **페이지 구성 5섹션 고정**: 소개 / 커리큘럼 / 기술스택 / 기대효과 / 커뮤니티 순서 준수
- **로고**: 텍스트 로고 `AI 바이브웹` — "웹" 글자만 코랄 컬러로 포인트 처리
- **GNB 위치**: 로고 왼쪽, 메뉴 오른쪽 정렬. 마지막 항목은 코랄 CTA 버튼
- **그래픽 이미지**: 외부 이미지 없이 SVG로 직접 제작한 일러스트 3종 삽입
- **글씨 안 잘림**: 모든 텍스트 `overflow: visible`, 고정 높이 컨테이너 사용 금지
- **CTA 이메일 폼**: 커뮤니티 섹션 하단 다크 배너에 인라인 이메일 입력 + 수강신청 버튼 배치
- **폰트**: `Playfair Display` (헤드라인, 세리프 잡지 느낌) + `Plus Jakarta Sans` (본문, 가독성)
